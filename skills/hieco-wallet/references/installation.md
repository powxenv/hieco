# Installation

Canonical docs:

- [`@hieco/wallet` installation](https://github.com/powxenv/hieco/tree/main/packages/wallet)
- [`@hieco/wallet-react` installation](https://github.com/powxenv/hieco/tree/main/packages/wallet-react)
- [`@hieco/react` installation](https://github.com/powxenv/hieco/tree/main/packages/react)

## Headless Wallet Runtime

```bash
npm install @hieco/wallet
```

```bash
pnpm add @hieco/wallet
```

```bash
yarn add @hieco/wallet
```

```bash
bun add @hieco/wallet
```

## React Wallet Layer

```bash
npm install @hieco/wallet @hieco/wallet-react
```

```bash
pnpm add @hieco/wallet @hieco/wallet-react
```

```bash
yarn add @hieco/wallet @hieco/wallet-react
```

```bash
bun add @hieco/wallet @hieco/wallet-react
```

Host app peer runtime:

- `react >= 18`
- `react-dom >= 18`

## Pair With `@hieco/react`

```bash
npm install @hieco/react @hieco/sdk @tanstack/react-query
```

```bash
pnpm add @hieco/react @hieco/sdk @tanstack/react-query
```

```bash
yarn add @hieco/react @hieco/sdk @tanstack/react-query
```

```bash
bun add @hieco/react @hieco/sdk @tanstack/react-query
```
